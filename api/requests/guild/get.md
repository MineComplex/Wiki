# GET /guild/:query
Получить информацию о гильдии

:query - поиск по префиксу или названию гильдии

Пример ответа:
```
{
    "name": "Админская",
    "prefix": "777",
    "color": "&4",
    "description": "Оффициальная админская банда",
    "createdAt": 1696251636727
}
```
