# GET /store/:server/categories
Получить список категорий режима

Пример ответа:
```
[
    {
        "id": "ranks",
        "server": "GLOBAL",
        "title": "Привилегии",
        "products": [
            "sapphire",
            "emerald",
            "diamond",
            "ruby",
            "amethyst"
        ]
    },
    ...
]
```
