# GET /store/categories
Получить список всех категорий

Пример ответа:
```
[
    {
        "id": "ranks",
        "title": "Привилегии",
        "products": [
            "sapphire",
            "emerald",
            "diamond",
            "ruby",
            "amethyst"
        ]
    },
    ...
]
```
