# GET /store/category/:id
Получить информацию о товаре

Пример ответа:
```
{
    "id": "ranks",
    "title": "Привилегии",
    "products": [
        "sapphire",
        "emerald",
        "diamond",
        "ruby",
        "amethyst"
    ]
}
```
