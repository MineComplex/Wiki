# GET /store/product/:id
Получить информацию о товаре

Пример ответа:
```
{
    "id": "emerald",
    "title": "Эмеральд",
    "image": "/content/image/ranks/emerald.png",
    "description": [
        "Норм донат"
    ],
    "price": 5000.0
}
```
